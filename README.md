🚀 COMBOLIST SPLITTER PRO
Advanced Large Archive Stream Processor | Production Ready

🌟 Giới thiệu & Tính năng
Công cụ chia nhỏ file RAR khổng lồ cực nhanh – Không cần giải nén trung gian, tối ưu cho xử lý dữ liệu lớn.

⚡ Siêu tốc độ: Streaming trực tiếp dữ liệu từ file RAR vào Python, bỏ qua bước giải nén thủ công tốn thời gian.

💾 Tiết kiệm ổ cứng: Không cần để trống hàng trăm GB để giải nén file gốc.

🛡️ Bảo vệ SSD: Cơ chế ghi cuốn chiếu thông minh giúp giảm thiểu áp lực lên ổ đĩa.

🔄 Chế độ Resume: Chia dở? Chỉ cần nhập số Part cũ, tool tự động nhảy cóc (Skip) đến vị trí đó để chạy tiếp.

⏹️ Tạm dừng thông minh: Tự động dừng sau mỗi đợt (Batch) để bạn kịp Move file đi chỗ khác hoặc Up lên mây.

📋 Line Integrity: Đảm bảo không bao giờ bị cắt ngang dòng Combolist.

🛠️ Yêu cầu & Cài đặt
Cài đặt thư viện: pip install rich 📦

Yêu cầu phần mềm: Máy đã cài sẵn WinRAR 🛠️

Cách chạy: Mở CMD/Terminal gõ: python split.py ▶️

📖 Hướng dẫn sử dụng nhanh
Source RAR: Nhập thư mục đầu vào file rar 📁

Output: Chọn thư mục lưu file con 📂

Size: Nhập dung lượng mỗi Part (Ví dụ: 1GB hoặc 500MB) 📏

Batch: Số Part chia xong thì dừng (Nhập 9999 để chạy một mạch không nghỉ) ♾️

Start Part: Bắt đầu từ Part mấy (Mặc định là 1) 🏁
