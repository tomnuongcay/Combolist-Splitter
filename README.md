🌟 Overview & Features
Combolist Splitter is a high-performance CLI tool designed to process and split massive RAR archives (billions of lines) without full extraction.

⚡ Ultra-Fast Streaming: Directly pipes data from RAR archives into Python, bypassing time-consuming manual extraction.

💾 Disk Space Efficiency: No need to reserve hundreds of GBs for extracted source files.

🛡️ SSD Longevity: Intelligent write-buffer mechanism reduces unnecessary disk wear.

🔄 Resume Capability: Interrupted? Simply enter the last Part number, and the tool will auto-skip to that position.

⏹️ Smart Batching: Automatically pauses after a set number of parts, allowing you to move data or clear disk space.

📋 Line Integrity: Guarantees no line-breaks within your combolist entries.

🛠️ Requirements & Installation
Python Library: pip install rich 📦

Software: WinRAR must be installed (to utilize UnRAR.exe) 🛠️

Run: Open CMD/Terminal and type: python split.py ▶️

📖 Quick Start Guide
Source RAR: Drag and drop your RAR file into the terminal 📁

Output: Select the destination directory for split files 📂

Size: Enter size per part (e.g., 1GB or 500MB) 📏

Batch: Number of parts before pausing (Enter 9999 for Continuous Mode) ♾️

Start Part: Starting part number (Default is 1) 🏁
